react-native之使用Icon

发表于 2017-03-08 | 阅读次数: | 分类于 react-native |

  今天在使用react-native的时候碰到一个问题，就是在TabBar碰到的。

情景

   在TabBar插入icon的时候，刚刚开始是放图片，后来发现不能控制图片的大小，找了很多方法都不行。

解决问题

跟据问题去google，发现一个库叫react-native-icons，但尝试了一下，一直报错，

由于我没仔细看readme，这个库都已经停止维护了。白白浪费了这么多时间，不过也有经验可取，知道了后继的添加库的方法。。
根据官方提示，建议去用react-native-vector-icons，然后就去配置xcode。

xcode配置

  1. 用xcode打开项目，把要用到的字体文件全部都放到fonts/里面，一般用到的字体文件是放在/node_modules/react-native-vector-icons/Fonts里面的，

    接着找到info.plist。添加字体文件，当然你也可以把自己的字体文件添加进去，见图中的第一个字体文件。

  1. 因为后继我们要用到TabBar，所以还要配置一个库，打开Libraries目录，会发现里面有很多.xcodeproj文件，然后点添加。把/node_modules/react-native-vector-icons/RNVectorIcons.xcodeproj添加进去

  2. 接着，点自己的项目名，然后选择Build Phases，然后里面有Link Binary With Libraries，把libRNVectorIcons.a添加进去。就可以了。具体操作，看下图。
